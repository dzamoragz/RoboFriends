RoboFriends es una aplicación construida con React que muestra un directorio de robots generados aleatoriamente. Incluye una barra de búsqueda que permite filtrar los robots por nombre en tiempo real. Este proyecto está diseñado como una práctica para reforzar conceptos clave de React como componentes, props, state, ciclos de vida y event handling.

✨ Características

✅ Lista de robots generados dinámicamente
✅ Filtro en tiempo real mediante barra de búsqueda
✅ Componentes reutilizables
✅ Diseño responsivo
✅ Uso de APIs para generar datos
✅ Construido con React + Vite / Create React App (ajustable)


🛠️ Tecnologías utilizadas

React
JavaScript (ES6+)
CSS / Tachyons (si usas Tachyons)
API de Robohash para generar imágenes de robots
API JSONPlaceholder (si usas datos de usuarios)


🚀 Instalación y uso
1. Clonar el repositorio
Shellgit clone https://github.com/tu-usuario/robofriends.git
cd robofriends
3. Instalar dependencias
Shelnpm install
4. Ejecutar el proyecto en modo desarrollo
Shellnpm start
La app estará disponible normalmente en:
http://localhost:3000
